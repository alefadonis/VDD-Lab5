Projeto de Visualização Interativa de Dados

Visualização dos ganhos pela venda de produtos do Varejo de 2022 a 2023

Breve resumo sobre os dados (e as histórias) visualizadas.

Dicas para interagir com a visualização:

  • Passe o mouse sobre os blocos e fluxos para ver mais detalhes sobre as conexões e os valores.
  • Clique nos elementos para destacar suas conexões com outros blocos e visualizar os dados adjacentes.
  • Use o recurso de zoom no navegador para ver detalhes específicos das áreas com mais conexões.
  • Redimensione a janela para ajustar automaticamente o gráfico ao tamanho da tela.
  • Se houver uma legenda ou ícones, utilize-os para filtrar ou destacar informações específicas.
  • Explore diferentes níveis de fluxo no gráfico para entender como os produtos estão interligados.
